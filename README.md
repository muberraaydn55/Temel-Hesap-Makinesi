Temel Hesap Makinesi (C++)

Bu proje, C++ programlama dilini öğrenirken geliştirdiğim ilk uygulamadır. Kullanıcının terminal üzerinden temel matematiksel işlemleri yapabilmesini sağlar.

Özellikler:
Dört İşlem Desteği; Toplama, Çıkarma, Çarpma ve Bölme.
Hata Kontrolü: Sıfıra bölme hatası gibi durumlar için basit kontrol mekanizması.
Kullanıcı Dostu: Anlaşılır terminal komutları ve yönlendirmeler.

Kullanılan Teknolojiler:
Dil: C++
IDE: Visual Studio

Nasıl Çalıştırılır?
1. Bu projeyi bilgisayarınıza indirin (Clone).
2. Herhangi bir C++ derleyicisi veya Visual Studio ile `Temel Hesap Makinesi.cpp` dosyasını açın.
3. Projeyi derleyin ve çalıştırın.
4. Terminaldeki talimatları izleyerek sayıları ve operatörü girin.
